## Overview
AI Health Predictor is a machine learning web application that analyzes patient data to assess risk factors and provide real-time health predictions.  
It combines a Python backend with modern ML frameworks and an interactive Streamlit frontend, deployed live for easy access.

---

## Features
- Predictive modeling using **Scikit-learn** and **TensorFlow**
- Interactive **Streamlit UI** for real-time predictions
- Secure API endpoints with **Flask/FastAPI**
- Dockerized for easy deployment
- Automated **MLOps pipeline** on AWS/GCP for training, versioning, and deployment

---

## Tech Stack
- **Languages:** Python, JavaScript
- **Frameworks:** Flask, FastAPI, React, Streamlit
- **ML Tools:** Scikit-learn, TensorFlow
- **Cloud/DevOps:** AWS, Docker, CI/CD
- **Database (optional):** MySQL for patient records
